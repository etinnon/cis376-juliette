Many different websites and forms use the following format above when it comes to entering in dates, years, and months.
This type of format does not allow the user to simply click the desired date within the first page.
On many websites, you have to click through the arrows to find the desired month or year.
This leads to a bad UX experience, as a user may have to click several times to find the date desired, which takes up so much more time than just simply providing the user the info they need all within the first page.
Navigate to the Improved design tab to see how I would suggest to fix this issue.
